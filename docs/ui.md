# UI

## Menu bar

- Status item is a compact chip, not text.
- Top lane represents active Node projects.
- Bottom lane represents watched ports currently occupied.
- A small warning dot appears when a watched port is busy and not owned by Node.

## Popover

The popover is a quiet card-based layout:

1. Header summary with counts and last updated time.
2. Watched-port strip with free, busy, and conflict states.
3. Grouped project cards for Node-family listeners.
4. Optional “Other listeners” section for non-Node owners.

## Settings

- `General`: launch at login, cadence, terminate confirmation.
- `Display`: grouping, watched ports, visibility of non-Node listeners.
- `About`: app info, diagnostics, snapshot export.
